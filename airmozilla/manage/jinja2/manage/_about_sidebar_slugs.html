About using the sidebar_ URL

If you want to use this static page for use in the sidebar, set the URL to something like this: sidebar_LOCATION_CHANNELSLUG.
For example, sidebar_top_how-tos.

Alternatively, you can just set the slug part to be * like this for example: sidebar_bottom_*. Then it will appear on every channel page. Specific channel slugs trumps the *.
